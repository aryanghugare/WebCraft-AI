# WebCraft AI (Client)

This folder contains the WebCraft AI front-end (React + Vite + Tailwind).

For the overall project description and structure, see the repository root README.

## Scripts

```bash
npm install
npm run dev
```

Other useful commands:

```bash
npm run build
npm run lint
npm run preview
```

## Notes

- The UI is currently scaffolded; API calls for AI generation / saving projects are not wired up yet.
- Some pages are placeholders while the backend is being integrated.
